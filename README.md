Based on regular slicer project. Added text field as type password for ***, and logic for searching changes from 'Contain" to "IS" to enforce password match record in dimension/fact tables.

Running the project
> npm install
> pbiviz start


Enable developer mode in Power BI service then use custom visual for development.

Reference pbiviz: https://learn.microsoft.com/en-us/power-bi/developer/visuals/develop-power-bi-visuals


**For certificate error try accessing https://localhost:8080/assets in browser
